# LinkRoom - 跨设备实时共享工作空间

在 Mac、Windows、iPhone、Android 等设备之间快速互联，传递文件和文本。

## 快速启动

```bash
cd linkroom
npm install --strict-ssl=false
npm start
```

启动后会显示局域网地址，其他设备在浏览器中打开该地址即可使用。

## 功能

- **创建/加入工作空间** — 一个设备创建房间，其他设备通过 6 位邀请码或扫描二维码加入
- **文本传输** — 实时发送文本消息，所有设备同步可见
- **剪贴板同步** — 一键发送剪贴板内容到工作空间
- **文件传输** — 拖拽或选择文件发送，支持最大 100MB
- **设备发现** — 自动显示工作空间中的所有在线设备及其类型
- **传输历史** — 房间内所有消息和文件记录在线可查

## 使用方式

1. 在一台设备的浏览器中打开 `http://<你的局域网IP>:3000`
2. 点击「创建工作空间」
3. 将显示的 6 位邀请码或二维码分享给其他设备
4. 其他设备打开同一地址，输入邀请码加入
5. 开始分享文本和文件
